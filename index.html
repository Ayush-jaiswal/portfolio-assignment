About my Projects

  • RFID based Car Parking System

    It is a smart paid car parking system based on RFID tag and is built on Arduino Platform.For each vehicle a RFID tag is allocated which is unique and at the time of entry and exit of the parking the owner just need to put the tag close to the RFID reader without any manual work. It also provides a recharging facility of the card using a Bluetooth based app.

  • GSM based theft detection System

    It is smart system which detects the theft of the vehicle and alerts the owner via SMS.This system can be operated in two modes i.e. User Mode and Theft Mode. When the user leave the vehicle for the parking he/she should leave the vehicle in theft mode. When the theft is detected the system will alert the owner and the user can switch off the vehilce remotely via SMS also. This system can also detect gas leakage and fire and alert the owner.
